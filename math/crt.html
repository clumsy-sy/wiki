中国剩余定理 CRT

若在序列 {ni​} 中任意两个互质，求方程组

⎩⎨⎧​x≡a1​x≡a2​x≡ak​​(modn1​)(modn2​)⋮(modnk​)​

的解。注意到，对于第 i 个方程，我们可以构造数

mi​=ni​1​j=1∏k​nj​

再求 mi​ 对 ni​ 逆元，则此式具有性质

(mi​mi−1​)modnj​={1,0,​j=ij=i​

这说明每一个解都不影响其他方程，可以独立求解再求和。因此有

ai​mi​mi−1​≡ai​(modni​)

对所有的解累加求和得到方程组的解

x≡i=1∑k​ai​mi​mi−1​(modj=1∏k​nj​)

int crt(int *aa, int *nn) {
    int prod = 1;
    int ret = 0;
    for (int i = 1; i <= n; i++)
        prod *= nn[i];
    for (int i = 1; i <= n; i++) {
        int m = prod / nn[i];
        ret += aa[i] * m * inv(m, nn[i]);
        ret %= prod;
    }
    return ret;
}
